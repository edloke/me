l'histoire

L'Histoire is an eBook app that allows readers to be able explore and read through a wide collection of different books and novels. This app aims to provide a one stop solution for readers to be able to store their existing library as well in the app as well as adding new books to read into their collections through the app.

Lo-Fi Prototype Hi-Fi Prototype

The color palette and Inria Serif font were chosen to create a consistent user interface that reflects the application’s essence, providing an experience similar to reading a physical book.

personas

my library
This page has been designed with several considerations in mind which is the main focus and essentials that could help users to continue where they left off from their previous reading.

This section is created to allow users to have quick access to their current readings and allow them to continue where they left off easily.

Access to the whole library, read list, history and bookmarks are placed in this section to allow users to easily browse through their collection. Graphical elements are also used to assist and improve accessibility.

The search button is placed on the top of the page to allow users to easily search through their library.

my account
During the process of designing this page, I have made several consideration on where to include the goals functionality that could help the user best keep track with their progress.

A goal summary is added to the account page to allow users to have quick view of their current goal progress which helps them to keep track of their goals.

The user's profile name and picture is displayed on the top of the page along with a customisable backdrop that helps user to personalise the application to their own preference.

refinements #1

The main updates involve changing the search button to a search bar, enhancing user cognition by clearly indicating that they can browse their library through this feature. This adjustment makes it easier for users to navigate and find content more intuitively.

Additionally, the book tile now shows the chapter where the user last left off, rather than just indicating that the book is in progress. The progress bar has also been moved to the bottom of the tile to create a less distracting experience when revisiting books.

refinements #2

One change is the introduction of a card-style layout to display goals, with added flexibility to view all previous goals from that page. Additionally, a backdrop behind the card and a greeting message create a more welcoming and enjoyable user experience.

Another update includes the addition of a user’s achievements tab at the top right, next to their profile picture. This tab allows users to view their current achievements and check the requirements for earning new ones.

accessibility

In designing the interface, I prioritized the primary task—reading. To enhance accessibility, I included a text size adjustment feature, allowing users to customize their reading experience. Another key consideration was the heading hierarchy, ensuring logical placement with larger, bold text preceding smaller text for clarity.

going forward
From this project, I found that there are many things to consider when designing a user interface such as the number of features and the userflow of each feature.

Hence, there must be planning that have to be done before starting designing such as what are the different user flows that would exist in the application and how it should be implement and if it could provide a better user experience to the user.
